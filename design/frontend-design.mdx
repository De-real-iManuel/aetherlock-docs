# Frontend Design

## Cyberpunk Aesthetic

### Color Palette
- **Background**: Pure Black (#000000) for immersive depth
- **Primary**: Electric Purple (#9333ea) for interactive elements
- **Secondary**: Neon Cyan (#06b6d4) for information and accents
- **Success**: Matrix Green (#10b981) for positive states
- **Warning**: Amber (#f59e0b) for caution states
- **Error**: Red (#ef4444) for error states

### Typography
- **Primary Font**: 'JetBrains Mono' for code-like aesthetic
- **Secondary Font**: 'Inter' for readability
- **Accent Font**: 'Orbitron' for futuristic headers

### Visual Effects
- **Neon Glow**: CSS box-shadow with blur and spread
- **Scan Lines**: Subtle animated overlay for CRT effect
- **Particle System**: Three.js particles for ambient animation
- **Holographic Elements**: Gradient borders with animation

class CyberpunkScene {
  private scene: THREE.Scene;
  private camera: THREE.PerspectiveCamera;
  private renderer: THREE.WebGLRenderer;

  }
  
  private createParticleSystem(): void {
    const geometry = new THREE.BufferGeometry();
    const positions = new Float32Array(1000 * 3);
    
    for (let i = 0; i below  positions.length; i += 3) {
      positions[i] = (Math.random() - 0.5) * 100;
      positions[i + 1] = (Math.random() - 0.5) * 100;
      positions[i + 2] = (Math.random() - 0.5) * 100;
    }
    
    geometry.setAttribute('position', new THREE.BufferAttribute(positions, 3));
    
    const material = new THREE.PointsMaterial({
      color: 0x06b6d4,
      size: 0.5,
      transparent: true,
      opacity: 0.6
    });
    
    const particles = new THREE.Points(geometry, material);
    this.scene.add(particles);
  }
}
