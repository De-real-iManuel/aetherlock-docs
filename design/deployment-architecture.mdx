# Deployment Architecture

## Development Environment
- **Blockchain**: Solana Devnet, ZetaChain Athens Testnet
- **AI Services**: AWS Bedrock (us-east-1)
- **Storage**: IPFS via Web3.Storage
- **Frontend**: Vercel deployment with preview branches
- **Monitoring**: AWS CloudWatch and Sentry integration

## Production Environment
- **Blockchain**: Solana Mainnet, ZetaChain Mainnet
- **AI Services**: Multi-region AWS Bedrock deployment
- **Storage**: Distributed IPFS with multiple pinning services
- **Frontend**: Global CDN with edge caching
- **Security**: WAF, DDoS protection, and security monitoring

## Infrastructure Components
- **Load Balancers**: AWS ALB for API traffic distribution
- **Databases**: PostgreSQL for metadata, Redis for caching
- **Message Queues**: AWS SQS for async processing
- **Monitoring**: Comprehensive logging and alerting
- **Backup**: Automated backups and disaster recovery
